# GeneVariantAnalyzer

GeneVariantAnalyzer is a Python and Biopython-based bioinformatics tool that compares DNA sequences from FASTA files and detects genetic variants, including SNPs, insertions, and deletions.

## Features

* FASTA file processing
* DNA sequence comparison
* SNP detection
* Insertion and deletion detection
* Mutation reporting

## Technologies

* Python
* Biopython
* Pandas
